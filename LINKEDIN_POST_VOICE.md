---

הדבר שהכי מרגיז אותי בחיפוש עבודה זה לא הדחיות.

זה להגיע לראיון לא מוכן.

לדעת שאתה הבן אדם הנכון לתפקיד — ולצאת מרגיש שלא הצגת את עצמך נכון.

אז הוספתי למערכת חיפוש העבודה שלי משהו חדש:

**ראיון קולי עם AI — בעברית או אנגלית, מותאם לשלב ולמראיין.**

אתה בוחר:
- שפת הראיון (🇮🇱 / 🇬🇧)
- שלב הראיון (HR / מקצועי / מנהל / סופי...)
- שם המראיין/ת מהסבבים הקיימים

ואז מתחילה שיחה קולית אמיתית.
Claude שואל שאלה. אתה עונה בקול. הוא מקשיב, מעכל, ועונה בקול.
עם פידבק אחרי כל תשובה: מה חזק, מה לשפר, ניקוד 1-10.

לא צ׳אט. לא טקסט. שיחה.

---

הדבר שגיליתי:
לדבר בקול זה שונה לגמרי מלכתוב.
כשאתה מדבר — אתה שומע את עצמך. שומע איפה אתה מהסס, איפה אתה חוזר על עצמך, איפה אתה לא משכנע.

3 סבבי תרגול לפני ראיון אמיתי שינו לי לגמרי את האנרגיה שאיתה אני נכנס.

## גרסה ב׳ — יותר טכנית / מקצועית

---

שדרגתי את מערכת חיפוש העבודה שבניתי.

הפעם הוספתי משהו שלא ציפיתי שיעבוד כל כך טוב:

**ראיון קולי אינטראקטיבי — STT + TTS + Claude בלולאה.**

הזרימה:
1. בוחר שלב ראיון + שפה + שם המראיין
2. Claude שואל שאלה בקול (ElevenLabs — קול אנושי אמיתי)
3. אתה עונה בקול → הדפדפן מתמלל
4. Claude מאזין → נותן פידבק → שאלה הבאה
5. אחרי 6-8 שאלות: סיכום עם ציון ממוצע

מה שמפתיע: הבדל עצום בין לכתוב תשובה לבין לדבר אותה בקול.
כשמדברים — מזהים הרבה יותר מהר איפה התשובה רופפת.

---

**Stack:**
- Anthropic Claude Sonnet — שאלות + פידבק
- ElevenLabs multilingual v2 — TTS אנושי
- Web Speech API — STT (Chrome/Edge)
- GitHub Pages — hosting
- עלות: ~$5-10/חודש (Anthropic) + ElevenLabs חינמי
