# בניתי לעצמי מנוע חיפוש עבודה עם AI — והנה הפרומפט שיבנה לך אחד תוך 10 דקות

לפני שבועיים התחלתי לחפש עבודה. כמו כל מי שעוסק ב-RevOps, השאלה הראשונה שלי לא הייתה "איפה אני מגיש" אלא **"איך אני בונה מערכת שתעשה את זה בשבילי"**.

אז בניתי אחת. סוכן AI שסורק משרות 3 פעמים ביום, מנקד כל אחת לפי הפרופיל שלי, ומעדכן טראקר חי — אוטומטית. תוך 10 ימים: 25 הגשות, 5 ראיונות, 3 שעברתי.

הדבר הכי טוב? **אתה יכול לבנות לעצמך אחד תוך 10 דקות** — בלי לכתוב שורת קוד. רק להדביק פרומפט אחד ל-Claude Code.

---

## מה המערכת עושה

- 🔍 **מוצאת משרות** — Claude סורק LinkedIn, Greenhouse, Lever, Ashby ומחזיר לינקים ישירים
- 🎯 **מנקדת התאמה 1–10** — לפי הפרופיל, המיקום והעדפות שלך
- 📊 **טראקר חי** — סטטוסים, סבבי ראיונות, הערות, אנליטיקה — באתר שמתעדכן לבד
- 🧠 **לומדת ממך** — מסמן משרה "לא רלוונטי"? היא לא תוצע שוב
- ✉️ **מייצרת מכתבים, קו״ח מותאמים והכנה לראיון** — בלחיצה
- 🎙️ **ראיון קולי אמיתי** — שיחה קולית עם AI, מותאמת לשלב ולמראיין, בעברית או אנגלית
- 💙 **תמיכה רגשית** — מנטור שמקשיב כשהתהליך כבד
- ☀️ **שאלת בוקר אישית** — כל כניסה מתחילה עם שאלה שמחזירה אותך למרכז

הכל רץ על ה-GitHub שלך, עם הדאטה שלך, בחינם (חוץ מ-~$5–10 לחודש על ה-API).

## הפיצ'ר שהכי גאה בו: ראיון קולי אמיתי

הדבר שהכי מרגיז אותי בחיפוש עבודה זה לא הדחיות. זה להגיע לראיון לא מוכן. לדעת שאתה הבן אדם הנכון לתפקיד — ולצאת מרגיש שלא הצגת את עצמך נכון.

כפתור **"תרגול ראיון"** בכל כרטיס משרה פותח setup של 3 שניות:

1. **שפה** — עברית 🇮🇱 או אנגלית 🇬🇧 (כי חלק מהראיונות מתנהלים באנגלית)
2. **שלב** — טלפוני / HR / מקצועי / מנהל / טכני / סופי — מתמלא אוטומטית מהסבבים שלך
3. **המראיין/ת** — שם ותפקיד, כדי שהשאלות יהיו מותאמות לזווית שלהם

ואז אפשר לבחור: **צ׳אט** או **ראיון קולי**.

בראיון קולי — Claude מדבר בקול. אתה עונה בקול. הוא מקשיב, מעכל, ועונה בקול עם פידבק אחרי כל תשובה: 💪 מה חזק · 🎯 מה לחדד · 📊 ניקוד 1–10. אחרי 6–8 שאלות: סיכום מלא עם ציון ממוצע ו-2 דברים לתרגל לפני הראיון האמיתי.

מה שגיליתי: לדבר בקול זה שונה לגמרי מלכתוב. שומע את עצמך. שומע איפה אתה מהסס, איפה אתה חוזר על עצמך, איפה אתה לא משכנע. 3 סבבי תרגול לפני ראיון שינו לי לגמרי את האנרגיה שאיתה אני נכנס.

> **טכנית:** STT דרך Web Speech API (Chrome/Edge) + TTS דרך ElevenLabs (קול אנושי, multilingual, חינמי לשימוש רגיל) + Claude Sonnet בלולאה לשאלות ופידבק.

## הפיצ'ר שלא ציפיתי שיהיה חשוב: תמיכה רגשית

חיפוש עבודה הוא תהליך שמרסק ביטחון. דחייה אחרי דחייה, שתיקה מחברות, הרגשה שאתה נמדד כל הזמן.

הוספתי כפתור 💙 — גלובלי (בכל עת) ופר-משרה (לאחר ראיון שלא הלך טוב).

הוא פותח שיחה עם מנטור שמשלב: הקשבה רפלקטיבית + ריפריים קוגניטיבי + אסטרטגיה מעשית + ויסות אנרגיה. לא פסיכולוג, לא בוט — שיחה שמחזירה פרספקטיבה ומוציאה אותך מהראש לפני הראיון הבא.

## שאלת הבוקר

כל פעם שנכנסים לטראקר לראשונה ביום, נפתחת שיחה קצרה — שאלה אישית שמתחלפת כל יום:

> "מה למדת על עצמך מהראיונות האחרונים?"
> "מה ההישג שאתה הכי גאה בו שתרצה לספר עליו בראיון הבא?"
> "מה אתה עושה טוב יותר מרוב האנשים בתפקידך?"

זה לא מרצה. זה שיחת מנטור-חבר של 2 דקות שמחזירה אותך למרכז לפני שמתחילים את היום.

## איך בונים — 4 צעדים

### צעד 1: פותחים חשבון Anthropic API
נכנסים ל-`console.anthropic.com`, מוסיפים $10 קרדיט, ומייצרים API key.
(זה לא Claude Pro — זה ה-API, pay-as-you-go. $10 מספיקים לחודש שלם.)

### צעד 2: מדביקים את הפרומפט הזה ל-Claude Code

```
אני רוצה שתבנה לי מערכת אוטומטית ומלאה לחיפוש עבודה, מבוססת Claude API,
פרוסה על GitHub Pages. עבוד בשלבים, וקרא לי לפני כל החלטת ארכיטקטורה.

== שלב 0 — אונבורדינג (שאל אותי לפני שאתה בונה) ==
שאל אותי אחת-אחת:
1. אילו תפקידים אני מחפש?
2. איפה אני מוכן לעבוד? (מיקום / רימוט)
3. מה הניסיון והחוזקות שלי?
4. בקש שאעלה קורות חיים (PDF/TXT)
מהתשובות צור config.json + profile.md.

== ארכיטקטורה ==
run_pipeline.py:
  search_jobs()  → Claude Opus + web_search → מוצא משרות פתוחות אמיתיות
  ats_scraper    → Greenhouse/Lever/Ashby API ציבורי
  score_jobs()   → Claude Haiku → מנקד 1-10 לפי הפרופיל
  filter         → ציון ≥4 + מיקום + פורסם ב-30 הימים האחרונים
  save           → scored_jobs.json (מקור אמת יחיד — לעולם לא מוחק משרות ידניות)
  build          → docs/index.html מתבנית
  git push       → GitHub Pages

== הטראקר (tracker_template.html) ==
HTML יחיד, RTL, mobile-first, Material 3, PWA, dark mode.
Layout דסקטופ: master-detail כמו Gmail (navigation rail + רשימה + פאנל פרטים).
לכל משרה כרטיס עם: ציון, סטטוס, סבבי ראיונות, הערות.

כפתורי פעולה — כל אחד פותח צ'אט חי עם Claude:
  - מכתב מקדים מותאם למשרה
  - הכנה לראיון (8 שאלות + נקודות דיבור + דגלים אדומים)
  - תרגול ראיון — setup עם בחירת שפה (עברית/אנגלית), שלב ומראיין.
    אפשרות צ'אט או ראיון קולי (STT + TTS). שאלה אחת בכל פעם,
    פידבק 💪🎯📊 אחרי כל תשובה, סיכום מלא אחרי 6-8 שאלות. טון מעצים.
  - תמיכה רגשית — מנטור שמשלב הקשבה, ריפריים, ואסטרטגיה
  - דוסייר חברה (מחקר, תרבות, חדשות)
  - קו״ח מותאם + PDF
  - "לא רלוונטי — למד מזה" → feedback.json → לא יוצע שוב

שאלת בוקר: בכל כניסה ראשונה ביום — שאלה אישית שמחזירה פרספקטיבה.

== עלויות ==
Opus לחיפוש (איכות), Haiku לניקוד (זול). חיסכון ~80%.
~$5-10 לחודש. צריך Anthropic API key (לא Claude Pro).
לראיון קולי אנושי: ElevenLabs API key חינמי (elevenlabs.io).

תתחיל עכשיו בשלב 0.
```

### צעד 3: מוסיפים מפתחות
- ב-GitHub → Settings → Secrets → Actions → `ANTHROPIC_API_KEY`
- בטראקר → הגדרות → Connections → מפתח Anthropic + (אופציונלי) מפתח ElevenLabs לקול אנושי

### צעד 4: מפעילים GitHub Pages
Settings → Pages → Branch: main → Folder: /docs. וזהו — יש לך טראקר חי בכתובת קבועה שמתעדכנת לבד.

## הערה על LinkedIn (חשוב)
אין API ציבורי למשרות LinkedIn מאז 2023, ו-scraping מנוגד ל-ToS. הפתרון החוקי שעובד: **חיפוש ציבורי** (Claude מוצא לינקים ישירים) + **ATS APIs** של Greenhouse/Lever/Ashby — שם רוב חברות ההייטק מפרסמות בלאו הכי.

## מה למדתי

**1. הפרדת מודלים = חיסכון של 80%.** Opus לחיפוש (איכות), Haiku לניקוד של עשרות משרות (מהירות וזול).

**2. מקור אמת יחיד.** קובץ JSON אחד שהפייפליין והטראקר קוראים ממנו. ברגע שהפרתי את הכלל — דאטה נמחק.

**3. מערכת AI נבנית מחדש אחרי שהיא קורסת.** הגרסה הראשונה שלי הראתה 14 משרות במקום 52. תיקנתי את באג השורש, הוספתי שכבת הגנה, וזה מה שעובד היום.

**4. לדבר בקול זה אחר לגמרי מלכתוב.** כשמתרגלים ראיון בטקסט — פספסת חצי. הקול חושף את ההיסוסים, את החזרות, את מה שלא מספיק משכנע. תרגול קולי לפני ראיון זה לא nice-to-have — זה הכנה אמיתית.

**5. חיפוש עבודה הוא תהליך GTM.** אתה המוצר. המגייסים הם השוק. מנהלים אותו עם מערכת — לא עם תחושת בטן.

**6. לא רק התהליך משתפר — גם הנפש.** מערכת שמחזירה שליטה מחזירה גם ביטחון. וביטחון הוא מה שמכריע ראיונות.
